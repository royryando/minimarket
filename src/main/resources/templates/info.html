LIST REST API


Produk

List semua produk [GET]

/produk/list

Cari berdasarkan nama produk/barang [GET]

/produk/cari-by-nama?nama=*nama*

Cari berdasarkan kode produk/barang [GET][PATH VARIABLE]

/produk/cari-by-kode/*kode*

Tambah dan Ubah produk/barang [POST][REQUEST BODY]

/produk/createupdate *data produk*

Hapus produk [POST][PATH VARIABLE]

/produk/delete/*kode*


Tipe Produk

List Semua tipe produk [GET]

/tipe/list

Cari berdasarkan kode [GET]

/tipe/cari-by-kode/*kode*

Cari berdasarkan nama [GET]

/tipe/cari-by-nama?nama=*nama*

Tambah dan Ubah Tipe produk [POST][REQUEST BODY]

/tipe/createupdate *data produk*

Hapus Tipe produk [POST][PATH VARIABLE]

/tipe/delete/*id*
